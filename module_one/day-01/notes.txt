ls - used to list everything in a directory
mkdir - used to create a directory
cd - used to enter that directory
ls takes optional flags - a to list all files and folders even hidden ones and -list all file info including the permissions
mkdir - to create a directory it takes the directory name as an argument
cd - to enter a directory it takes .. - to move back or name to go to a new one
touch - used to create a file , takes in the name of the file as an  argument
mv - used to move a file or directory and also with different args used to rename a file or directory ,  the first argument will always be the target being moved or renamed then if the second argument is a directory that exists it means move but if it is a name then it means rename it
cat used to append texts in a file
cat >> filename

git diff - to differentiate the working vs the staging area

git rm --cached - to unstage changes
